Detailed Description

This block is only needed when running lora_sdr in multi streaming mode (aka multiple transmitting chains) This block is intended to ensure all done signals from all the transmitting blocks are gathered and once all transmitting blocks are done, send the done signal to all transmitting blocks, and thus ensuring the entire flowgraph is closed correctly.
